# USB-Hub
I made this project because I need more usb ports on my computer and it sounded fun.
This is a usb hub that has two usb c upstream ports that can be switched between via a switch, as well as four downstream usb a ports that can be turned on and off via switch,  it is also equipped with esd protection on all of the downstream ports
